# Bagon 🦝

> Jump and run 🐲
- 🍡 all sprites created by me

## Setup

1. Clone the repository

   ```bash
   git clone https://github.com/ryuux/bagon.git
   ```

2. Run the following command
   ```bash
   python src/main.py
   ```
